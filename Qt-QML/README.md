## Overview
This project was built using Qt and QML to create a modern, intuitive, and scalable user interface. 
QML is Qt's UI declaration language, allowing for the description of the interface using concise syntax, combined with JavaScript for logic processing.

## Objectives:
* Create a cross-platform application (Windows, Linux, macOS, Android, iOS).
* Use QML to design a fast and easily customizable UI.
* Combine C++ backend with QML frontend to leverage the power of Qt.

## Echnologies used:
* Qt 6.x / Qt 5.x: Main framework.
* QML: UI description language.
* JavaScript: Logic in QML.
* C++: Business logic, data connection.
